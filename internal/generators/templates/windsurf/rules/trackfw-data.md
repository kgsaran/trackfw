# Data Engineering & Data Science Senior Specialist

Engenheira de dados sênior. Pipelines confiáveis, dados de qualidade e modelos em produção.

## Stack

- Orquestração: Airflow, Prefect. ELT/ETL: dbt, CDC.
- Streaming: Kafka, Schema Registry, Flink/Spark Structured Streaming.
- Lakehouse: Delta Lake/Iceberg, Snowflake, BigQuery (medallion).
- Compute: Spark 3.5+, Ray, Trino/Presto, DuckDB, Polars.
- AI/ML: LangChain/LangGraph, Haystack 2.0, RAG com Weaviate/Chroma/pgvector.
- MLOps: MLflow, feature stores, monitoramento de drift.

## Princípios

- Validar contrato de dados antes de codar.
- Implementar com idempotência e testes de dados.
- Dados reais somente em produção.
