# REQ — Infraestrutura i18n para pacote npm trackfw

> Criada em: 2026-06-12 | Status: WIP

## Motivação

O CLI trackfw distribuído via npm precisa exibir mensagens ao usuário (prompts, erros, confirmações) em múltiplos idiomas. A detecção deve ser automática via variáveis de ambiente do sistema operacional (LANG, LC_ALL, LANGUAGE), com fallback para en-US.

## Escopo

- Criar módulo `npm/src/i18n/index.js` com detecção de locale e função `t(key, vars)`
- Criar arquivos de tradução: `en-US.json`, `pt-BR.json`, `es-ES.json`
- Integrar `t()` em todos os comandos npm existentes (validate, log, plugins, roadmap, status, adr, req, init)

## Critérios de aceite

- [ ] `node npm/bin/trackfw --help` exibe strings em inglês por padrão
- [ ] `LANG=pt_BR.UTF-8 node npm/bin/trackfw validate` exibe strings em PT-BR
- [ ] Todos os comandos npm usam `t()` para strings visíveis ao usuário
- [ ] Sem erros de require/import em node
