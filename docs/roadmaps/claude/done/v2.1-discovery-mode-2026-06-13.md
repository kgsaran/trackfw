## Wave 3 — Bloco A: trackfw discover
> MLs sequenciais

### ML-3A — `internal/discover/discover.go`
**Status:** ✅ Concluído

**Arquivos:** `internal/discover/discover.go` (novo), `internal/discover/discover_test.go`

```go
type DiscoveryResult struct {
    ADRDirs            []string
    REQDir             string
    RoadmapDir         string
    RoadmapNamespacing string
    Agents             []string
    ADRCount, REQCount, RoadmapCount int
    HasTrackfwYAML, HasTrackfwLog bool
    GovernanceScore int  // 0-100
}

func Scan(rootDir string) (DiscoveryResult, error)
```

Lógica:
- ADRDirs: Walk `docs/adr/**/*.md`, agrupar por subdir não-vazia
- REQDir: testar `docs/req`, `docs/requisições`, `docs/requirements`
- Namespacing: se `docs/roadmaps/*/wip/` existe → by_agent
- Agents: subdirs de roadmapDir com pelo menos 1 estado válido (backlog/wip/done)
- Score: +20 por ADRs, REQs, roadmaps presentes; +20 por links ADR↔REQ; +20 por WIP sem violações

Testes: `TestScan_Empty`, `TestScan_Flat`, `TestScan_ByAgent`, `TestScan_CMDBLike`

Paridade npm: `npm/src/discover/index.js`
