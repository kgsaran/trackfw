# Roadmap: v2.7.0 — trackfw serve Dashboard Web (paridade Go · Node.js · Python)

> Criado em: 2026-06-14 | Status: 🔄 WIP

## Diagnóstico / Contexto

O `trackfw serve` hoje é um stub HTTP sem UI (Go apenas). A proposta é evoluir
para um dashboard web completo com paridade nos 3 CLIs — Go, Node.js e Python.

Os dados já existem: artefatos MD em disco, grafo de dependências via frontmatter,
log de transições em `.trackfw-log`. Node.js e Python já têm `parseLog`/`_parse_log`
e `calculate`/`_calculate` prontos em `commands/metrics.js` e `commands/metrics.py`
— reutilizar, não reescrever.

Assets web (HTML/JS/CSS) são **compartilhados** entre os 3 CLIs:
- Go: `embed.FS` (single-binary)
- Node.js: `npm/src/serve/static/` instalado com o pacote
- Python: `pypi/trackfw/serve/static/` via `package_data`
