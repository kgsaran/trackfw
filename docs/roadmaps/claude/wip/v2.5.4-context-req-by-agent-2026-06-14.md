# Roadmap v2.5.4 — Fix context REQ by_agent

> Criado em: 2026-06-14 | Status: 🔄 WIP

## Diagnóstico

Resíduo pós-v2.5.3: `trackfw context` exibe `## REQs (0)` em layout by_agent.
Os Roadmaps já são varridos corretamente em `context`; só a parte de REQs ficou flat.
Adicionalmente, `validateADRsAreReferenced` (Go) usa `os.ReadDir` flat.

**Pontos afetados:**

| Arquivo | Linha | Problema |
|---------|-------|---------|
| `internal/generators/context.go` | ~57 | `os.ReadDir(cfg.REQDir)` flat |
| `internal/validator/validator.go` | ~854 | `os.ReadDir(cfg.REQDir)` em `validateADRsAreReferenced` |
| `npm/src/commands/context.js` | ~102 | `collectEntries(reqDir, 'REQ')` flat |
| `pypi/trackfw/commands/context.py` | ~108 | `_collect_entries(req_dir, "REQ")` flat |

**Status:** ⬜ Pendente

**Arquivos afetados:**
- `internal/generators/context.go`
- `internal/validator/validator.go`
- `internal/validator/validator_test.go` (ou `internal/generators/context_test.go` se existir)

**Ações:**

1. Em `internal/generators/context.go`, substituir o bloco de coleta flat de REQs (linha ~57):
   ```go
   reqEntries, err := os.ReadDir(cfg.REQDir)
   ...
   ```
   Por varredura by_agent-aware, espelhando o padrão de Roadmaps já existente no mesmo arquivo
   (linha ~74). Se `cfg.RoadmapNamespacing == config.NamespacingByAgent`:
   - Descobrir agentes via `cfg.Agents` ou `os.ReadDir(cfg.REQDir)`
   - Para cada agente, para cada estado em `["backlog","wip","blocked","done","abandoned"]`:
     - `os.ReadDir(filepath.Join(cfg.REQDir, agent, state))`
     - Para cada `.md`: extrair status do frontmatter e append a `reqs`
   Senão: comportamento flat atual preservado.

2. Em `internal/validator/validator.go`, na função `validateADRsAreReferenced` (~linha 845):
   Substituir:
   ```go
   reqs, err := os.ReadDir(cfg.REQDir)
   ...
   for _, r := range reqs {
       if r.IsDir() { continue }
       b, err := os.ReadFile(filepath.Join(cfg.REQDir, r.Name()))
   ```
   Por uso de `resolveREQFiles(cfg)` (já existe desde v2.5.3):
   ```go
   reqPaths := resolveREQFiles(cfg)
   var allREQContent strings.Builder
   for _, p := range reqPaths {
       b, err := os.ReadFile(p)
       if err == nil {
           allREQContent.Write(b)
       }
   }
   ```

   - `TestContextREQByAgent`: cria estrutura `req_dir/claude/wip/req.md`, chama `GetContext` com

   const reqNamespacing = cfg.roadmapNamespacing || cfg.roadmap_namespacing || ''
   if (reqNamespacing === 'by_agent') {
     const STATES = ['backlog', 'wip', 'blocked', 'done', 'abandoned']
     let agents = cfg.agents || []
     if (!agents.length) {
       try { agents = fs.readdirSync(reqDir).filter(f => fs.statSync(path.join(reqDir, f)).isDirectory()) } catch (_) {}

   req_dir = cfg.get("req_dir", "docs/req")

   if namespacing == "by_agent":
       STATES = ["backlog", "wip", "blocked", "done", "abandoned"]

       if not agents:
           try:
